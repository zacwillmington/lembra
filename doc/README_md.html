    Lembra Welcome to Lembra. This application is a flashcard app that allows you to create an account, login and create decks that you can add cards to. Each card has two sides, one front for the question and back for the answer. You're also able to mark the card as learnt once you have memorised the answer.

    Usage You can view Lembra in action by heading to example.

    Development After cloning the repo, run bundle install then run shotgun to open up a browser.
